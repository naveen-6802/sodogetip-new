Quick Start for SoDogeTip

This quick-start will help reddit users to undestand how to use sodogetip on reddit easily.

Use website !

You can use the website to register your account, use login function and you will see an button to create your account


The reddit way !

To register, you need to send the +register to the bot.


To find your deposit address, send the +info command to the bot.


To tip a user, you need to comment a post and mention the bot using the following syntax: +u/sodogetip [amount] [currency] verify



You can also use some keywords instead of a fixed amount.

Use website !

You can use the website to register your email, connect you on website and go to settings, you will have link to register your email
